gpslogger-rails
===============

Rails backend for gpslogger

## Prerequisites
- rvm 1.23.10 (stable)
- ruby1.9.3-p362
- rails4.0.0
- postgresql 9.3
- rspec-core (2.14.6)
- rspec-expectations (2.14.3)
- rspec-mocks (2.14.4)
- rspec-rails (2.14.0)
